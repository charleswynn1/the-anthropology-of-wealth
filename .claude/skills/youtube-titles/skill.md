# YouTube Title Creation

## The One Rule That Matters Most

**5-7 words. Under 50 characters. No exceptions.**

Shorter titles process faster on screen, force tighter word choice, and leave room for the thumbnail to do its job. Aim for ~34 characters. The hook must land in the first half of the title — YouTube truncates with "..." on some devices, and viewers skim the first few words before deciding to read the rest.

## Open Loop Formats

Pick the format that fits the content. Use a different format for each variation so the options feel meaningfully distinct.

| Format | Example |
|--------|---------|
| **Counterintuitive** | "Why Studying Less Gives You an Unfair Advantage" |
| **Insider Information** | "Day in the Life of an AI Startup Founder" |
| **Challenging Convention** | "Why Californians Regret Moving to Austin" |
| **Numbers** | "These 11 Income Streams Made Me $220,000" |
| **Ask a Question** | "Where Did All the Bubble Wrap Go?" |
| **Contrast** | "$1 vs $500,000 Plane Ticket" |
| **Superlatives** | "I Tested the World's 5 Deadliest Weapons" |
| **Impossible Challenge** | "Can You Get the Splits in One Day?" |
| **Regrets** | "I Bought 22 Rental Units and Now I Regret It" |
| **Exciting Journey** | "I Trained Like a Black Belt for 90 Days" |

## Word Choice

Favor grounded power words — *unexpected, hidden, overlooked, surprising, exceptional* — over sensational ones like *insane, shocking, unbelievable, mind-blowing*. Sensational words consistently underperform in large-scale data. For history/documentary content, words like *secret, hidden, deadliest, "the truth about"* drive clicks disproportionately.

## Title Creation Process

1. **Find the one detail that stops a scroll** — not the topic, but the single most surprising fact, number, or moment from the video
2. **Pick an open loop format** from the table above
3. **Write the title in 5-7 words** with the hook in the first half. Check: under 50 characters?
4. **Generate 5-7 variations** using different formats. For each, note the format used, word count, and character count.
5. **Rank by loop strength** — which title makes you ask the most questions? Recommend a top pick and runner-up with a one-line reason for each.
6. **Check `youtube_titles.md`** for duplicates before finalizing

- Pattern: `[Specific dramatic hook] | [Event Name]`

### 60 Seconds of Chaos — Deep Dives (Longform)
- Pattern: `[Dramatic narrative hook] — [Event, Date]`
- Evocative, not clickbait. Reference the most compelling human element.
- Check `longform/youtube_titles.md` for existing titles

### All Channels
- Never use generic descriptions as titles ("History of Event X")
- Numbers, dates, and measurements create specificity ("173 Dead", "600 Feet Below")
- All temperatures in Fahrenheit
